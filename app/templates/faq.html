We all know that Indian goverment did not do much to stop the spread of the second wave of COVID-19. There have been a number of irregularities in the reporting the number of cases and deaths from covid. We believe the only way to make sure that all cases are counted is for people to report these cases. This website was created in an attemp to do that.

Currently we are accepting the following information:
  1. Number of infections
  2. Number of fatalities (deaths)
  3. Country [Supports only, India and Nepal]
  4. Pin code (your postal zip code)

As we collect more and more data we will produce summary and information graphics about the data. This will help cover gaps in goverment reporting of covid cases.
